# Pok-dex-Web
Pokédex Web é uma aplicação funcional que permite ao usuário buscar um Pokemon pelo nome ou ID e visualizar suas informações básicas diretamente na tela, consumindo dados em tempo real de uma API pública.

## Publicação

O projeto pode ser publicado com GitHub Actions no GitHub Pages. Depois do deploy, a URL fica neste formato:

https://amisterdan7.github.io/Pokedex-Web/

No GitHub, também é preciso deixar o Pages configurado para usar GitHub Actions em Settings > Pages.
